What'is Spotify URL? How to get it?

URL is a link to your playlist

You can optain it by:

  1. Go to the Spotify Playlist
  2. Right click or Click on 3dot icon
  3. Go to share -> Copy Spotify Link
What'is Spotify ID, How to get it?

ID is a unique string of numbers assigned to every playlist. In order to optain it follow these steps:

  1. Go to the playlist screen in Spotify
  2. Right Click or Click 3dot icon
  3. Go to share -> Copy Spotify URI

Please note that ID is only last string of characters in URI (bold portion in the example)paste ONLY this portion of the URI, example: spotify:playlist:2hU56mpHuOEAlyf64raVoG
